# Assets

This directory contains tracked public assets for Psychevo.

## Canonical Logo

`psychevo-logo.svg` is the canonical Psychevo logo asset. Use this file for
README, documentation, website, package, favicon, and social-preview derivation
unless a later brand spec replaces it.

Files under `.local/design/` are ignored local design exploration material.
Do not reference them from tracked documentation or package metadata.

When a target surface supports SVG, reference `assets/psychevo-logo.svg`
directly and use `Psychevo` as the default alt text. Generate raster previews,
favicons, and social-card artwork from this SVG rather than from older
candidates.
